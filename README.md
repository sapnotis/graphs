## КОМПОЗИЦИЯ ПРОЕКТА:
1) Модель игровой доски с прямоугольными блоками.
2) Граф вершин, отражающих состояние поля, его визуализация.
3) Исполняемый код, принимающий на вход начальное состояние поля и вызывающий окно с визуализацией графа состояний.

- Разработка доски, графа и итогового main ведётся в раздельных побочных ветках.

- Сборка происходит под линукс при помощи makefile.

Для сборки с элементами графа ( graph-components, tools, main-graph-tester ) должна стоять SFML ( на линуксе ставится строкой: sudo apt-get install libsfml-dev ). Не загружайте саму SFML в этот репозиторий.
