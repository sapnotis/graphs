# Граф состояний дискретной системы (головоломки klotski)

## Что делает

Превращает Систему в графическое окно с 3d графом, который можно поразглядывать под разными углами. Система - объект, описываемый набором целых чисел, причём не любой набор чисел является разрешённым. Разрешённое состояние получает вершину в графе. "Соседние" разрешённые состояния соединяются ребром. Здесь в качестве Системы выступает поле головоломки klotski.

## Основные файлы

- board - задаёт игровое поле, разрешает/запрещает набор чисел.
- combinator - ищет все разрешённые состояния, до которых можно добраться из начального, конструирует граф.
- graph-components - логика графа и вывод на экран.
- config.txt - позволяет задавать произвольное игровое поле.
- makefile - собирает проект (пока что сборка прозводится только g++ под linux).
